# Employee Management System - Tkinter

A Python desktop application for managing employees using Tkinter GUI and SQLite database.


## Features

- User Registration
- Secure Login System
- Dashboard
- Employee Management
- Department Management
- Attendance Tracking
- Salary Management
- Reports Export
- Settings
- Logout System


## Tech Stack

- Python
- Tkinter
- SQLite


## Project Structure


Employee_Management_System

- database
- pages
- assets
- exports


## Modules


### Authentication

Register and login users.


### Dashboard

View employee statistics.


### Employees

Add, update, delete and view employees.


### Departments

Manage company departments.


### Attendance

Track employee attendance.


### Salary

Calculate employee salary.


### Reports

Export employee data.


## Installation


Clone repository

git clone your-repository-link


Move into folder

cd Employee-Management-System-Tkinter


Install requirements

pip install -r requirements.txt


Run

python main.py
